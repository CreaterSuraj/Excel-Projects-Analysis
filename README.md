Sales Dashboard 1 (Microsoft Excel)
Project Overview

This project is an interactive Sales Dashboard built in Microsoft Excel to analyze retail sales performance and provide actionable business insights. The dashboard transforms raw transaction data into meaningful visualizations using Pivot Tables, Pivot Charts, Slicers, and KPI metrics, enabling users to monitor sales trends and make data-driven decisions.

The dataset contains over 1,000 retail transactions with information such as customer demographics, product categories, sales amount, quantity, and purchase dates.

Objectives
Analyze overall sales performance.
Track monthly sales trends.
Compare revenue across product categories.
Understand customer purchasing behavior by gender and age group.
Build an interactive dashboard for business reporting.
